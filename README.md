# Journ-Web

Static web client for Journ, a simple journal.


## Getting Started

1. Install

   ```
   git clone https://github.com/patgrasso/journ-web.git
   cd journ-web
   npm install
   npm start
   ```

2. Go to **Connect to Server**
3. Enter the address of a running Journ server
4. Hit **Connect**
5. Log in
